📖 User Manual

🚀 Getting Started

Welcome to N.A.M.O.R. x BUBT HUB Audio Transcription System! This powerful tool converts audio and video files into text with real-time typing animation.

🎥 YouTube Transcription

How to Use:

  1. Click on the "YouTube URL" tab
  2. Paste any YouTube video URL into the input field
  3. Click "Transcribe" button
  4. Watch as the transcription appears in real-time with typing animation!

Supported URLs:

  • Standard: youtube.com/watch?v=...
  • Short: youtu.be/...
  • Embedded: youtube.com/embed/...

📁 File Upload Transcription

Supported Formats:

  • Audio: MP3, WAV, M4A, AAC, OGG, FLAC, WMA
  • Video: MP4, AVI, MKV, MOV, FLV, WMV, WebM, MPEG, 3GP

How to Use:

  1. Click on the "Upload Audio/Video" tab
  2. Drag & drop your file OR click to browse
  3. Wait for automatic transcription to begin
  4. Read the text as it types out character-by-character!

✨ Features

  • Real-time Typing Animation: Text appears smoothly, keeping you engaged
  • Timestamped Segments: YouTube videos show time markers for easy navigation
  • Live Indicator: Visual feedback with pulsing green dot and sound waves
  • Progress Tracking: See exactly what stage the transcription is at
  • Copy to Clipboard: One-click copying of entire transcription
  • Download as TXT: Save transcriptions for later use

⚡ Tips for Best Results

  • Use clear audio with minimal background noise
  • Videos under 10 minutes work fastest
  • English language provides best accuracy
  • Ensure stable internet connection for YouTube videos
  • For large files, be patient - typing animation makes wait time enjoyable!

🎯 Understanding the Interface

  • 🟢 LIVE Badge: Indicates active transcription in progress
  • Progress Bar: Shows completion percentage
  • ⏱️ Timestamps: Pulsing badges show current segment being typed
  • Typing Cursor: Blinking cursor follows the text as it types
  • Copy Button: Instantly copy all transcribed text
  • Download Button: Save as .txt file

❓ Troubleshooting

Error: "Could not understand audio"

Solution: Audio may be unclear or have heavy background noise. Try a clearer recording.

YouTube download fails

Solution: Some videos are restricted. Try a different public video or use the file upload method.

Long processing time

Solution: Longer videos take more time. The typing animation keeps you entertained during the wait!
